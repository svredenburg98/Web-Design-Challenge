Summary: Latitude vs X

Visualizations

The purpose of this project was to analyze how weather changes as cities approach the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities. After assembling the dataset, we used Matplotlib to plot various aspects of the weather against latitude. Factors we looked at included: max templerature, humidity, cloudiness and wind speed. This site provides the data and visualizations created as a part of the analysis. as well as explanations of each visualizations.
